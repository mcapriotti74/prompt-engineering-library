### Philosophy
This release establishes the **CORE** principles of the Prompt Engineering Library:
- Generic, technology-agnostic practices
- Stable, long-lasting foundations
- Evidence-based development (NO SUPONER - VERIFICAR)
- Multi-agent orchestration
- Test-Driven Development (TDD)

### Ecosystem Vision
This repository serves as the **STABLE CORE** of a larger ecosystem:
- **prompt-engineering-library** (this repo): Generic principles, stable (v1.0 target: 6-12 months)
- **prompt-engineering-tools** (future): Specific tool catalogs, updated frequently
- **prompt-engineering-recipes** (future): Complete use case examples
